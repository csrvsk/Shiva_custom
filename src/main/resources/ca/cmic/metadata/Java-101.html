Ampersand Check

Checks for the use of the '&' symbol in code.

Rationale

Using the '&' symbol can sometimes make code less readable and harder to maintain. In addition, it can cause subtle bugs if not used correctly.

Impact

Using the '&' symbol can lead to code that is harder to read and maintain. It can also introduce subtle bugs if not used correctly.

Remediation

To fix this issue, replace the '&' symbol with '&&' as appropriate.
